# Thushani_2025_AT6

This project demonstrates user input validation in Python using a custom `UserInputValidator` class.

## Overview

The project includes:
- A validator class that checks for positive integers
- Demo scripts showing different usage scenarios
- Progressive commits demonstrating development stages

## Files

- `user_input_validator.py`: Core validation class with `validate_positive_integers` method
- `commit1_demo.py`: Basic demo with single validator instance
- `commit2_demo.py`: Demo creating two validator instances
- `commit3_demo.py`: Advanced demo validating data with both instances
- `extra_info.txt`: Additional project information
- `.gitignore`: Ignores `extra.txt` file

## Usage

Run the demo scripts in order:

```bash
python commit1_demo.py
python commit2_demo.py
python commit3_demo.py
```
